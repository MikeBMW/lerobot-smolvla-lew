## 六、Sys-0/1/2 延迟分解

```
端到端延迟链路:
  传感器采集 → Sys-0 安全检查 → Sys-1/Sys-2 推理 → 动作输出

  RealSense:      ~33ms (30fps)
  力传感器:       ~0.1ms (1kHz)
  Sys-0 检查:     <0.1ms
  ACT 推理:       0.5~8.4ms
  SmolVLA 推理:   215~300ms
  动作下发:       ~1ms (UDP)
  ─────────────────────────
  ACT 端到端:     ~35ms
  SmolVLA 端到端: ~250-330ms
```
