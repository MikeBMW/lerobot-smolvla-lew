### Z-MAX 工程映射

| 脑结构 | 功能 | Z-MAX模块 | 工程实现 |
|--------|------|-----------|----------|
| **枕叶**(V1-IT) | 视觉理解 | Sys-2 数据层 | LeRobot Dataset + 图像预处理 |
| **颞叶**(IT+语言区) | 语义+指令 | VLM Text Model | LLaMA 16层 × 960维 |
| **顶叶**(PPC) | 空间+动作映射 | Expert Cross-Attn | 交叉注意力读VLM |
| **额叶**(PMC+M1) | 运动执行 | Expert Self-Attn | 自注意力规划 |
| **基底节** | 动作选择 | Flow Match time_mlp | Linear(1440→720)+SiLU |
| **小脑** | 轨迹平滑 | Flow Match ODE | Euler积分×10步 |
| **丘脑** | 信号中继 | state_proj | Linear(32→960) |
| **胼胝体** | 左右脑通信 | Cross-Attention | VLM→Expert信息桥 |
| **海马体** | 经验记忆 | 训练Checkpoint | policy.pt持久化 |
