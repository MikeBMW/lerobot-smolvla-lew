## 五、Phase 0 — 你们拿到的第一台机器就是从这里起步的

Z-MAX 的研发分为多个 Phase。Phase 0 是我们的**工程验证阶段**。

这个阶段，Z-MAX 运行在我们深度合作供应商提供的**6-DOF 机械臂平台**上。供应商平台提供了基础的关节运动能力和 ROS2 驱动接口。我们在其之上构建了 Z-MAX 的三层架构：

```
┌────────────────────────────────────────┐
│  System 2 · 任务规划 (L4 大脑)          │  ← 未来升级
├────────────────────────────────────────┤
│  Sys-11 · 动作系统 (VLA模型)            │  ← L3 升级
│  Sys-12 · 引导系统 (世界模型)            │
├────────────────────────────────────────┤
│  System 0 · 硬件抽象层                  │  ← L2 基线
│  · 6-DOF 关节运动 · 夹爪力控           │
│  · 六维力感知 · RealSense 3D视觉        │
│  · 塔灯/急停/触觉/扫码 IO              │  
│  · 1ms EtherCAT 实时控制               │
└────────────────────────────────────────┘
         ↓ ROS2 Domain 23
    供应商机械臂平台 (运动执行)
```

**Phase 0 的意义**: 我们在供应商平台上验证了 Z-MAX 的 L2 级全部功能——精准运动控制、力控柔顺操作、安全急停、多工序自动执行。每一个功能都在真实光模块产线上跑过。

## 六、日常操作 — 5 分钟学会

### 6.1 开关机

```
开机: 按下主电源 → 等待塔灯变绿 → 打开 XSpace Studio
关机: Studio 里点"急停" → 断开主电源
```

### 6.2 运行工序

```
1. 在 XSpace Studio 首页选择工序:
   · 固件加载   · 老化上料   · 老化下料   · 模块测试

2. 点击"开始" → Z700 自动执行

3. 塔灯颜色:
   🟢 绿色 = 正常运行
   🟡 黄色 = 需要人工处理 (缺料/模块异常)
   🔴 红色 = 急停 (立即检查)
```

### 6.3 处理常见问题

| 现象 | 原因 | 操作 |
|------|------|------|
| 黄灯亮 | 上料盘空了 | 补充模块，点"继续" |
| 夹爪抓不住模块 | 模块位置偏移 | 点"重新定位" |
| 红灯亮 | 急停按下 | 排除障碍，拧开急停，点"复位" |
| 测试不通过 | 模块本身不良 | 模块放入不良品盒，继续下一个 |
