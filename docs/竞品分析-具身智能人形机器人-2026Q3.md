## 二、逐家详析

### 1. 开普勒机器人 (Kepler)

| 维度 | 详情 |
|:--|:--|
| **成立** | 2023年 |
| **产品** | 先行者K系列通用人形机器人 (K1→K2) |
| **特色** | 双臂协同搬运25-30kg；1小时充电续航8小时 |
| **应用** | 高空焊接、汽车制造、工业物流 |
| **资本** | 2026年5月被杭州柯林电气(688611)收购41.57%成为控股方 |
| **财务** | 2025年营收433.72万，净亏6693.9万；2026Q1营收263.63万，净亏1709.27万；在手订单4700万+ |
| **对标启示** | 工业落地先行者，但尚未盈利；被上市公司控股后资金充裕 |

### 2. 乐聚机器人 (Leju Robot)

| 维度 | 详情 |
|:--|:--|
| **成立** | 2016年3月，哈工大学生冷晓琨创建，注册142万 |
| **首席科学家** | 蔡鹤皋院士 |
| **产品线** | Aelos小型双足(30-70cm)、通用人形 |
| **核心能力** | 自研一体化关节 + 双足步态算法 |
| **发展路径** | Ph1教育(走进5000+学校完成原始积累)→ Ph2特种应用(医疗等)→ Ph3 toC家庭(健康陪护) |
| **产能** | 2026年3月佛山万台级产线(30分钟/台)；深圳中试产线(500-1000台/年) |
| **资本** | 2026年申请创业板IPO，预计2028年盈利 |
| **对标启示** | 教育→工业→家庭的清晰三阶段路径；万台产能行业领先 |

## 四、战略建议

1. **守住光模块赛道** — 目前无竞品深耕此垂类，先发优势窗口期约12-18个月
2. **对标星海图** — 其VLA模型+具身OS路线与Z-MAX的VTLA高度重合，需持续追踪
3. **借鉴乐聚** — 其万台产能经验可为Z-MAX量产提供参考
4. **关注智元/银河** — 估值最高、品牌最强，是生态位主要威胁
