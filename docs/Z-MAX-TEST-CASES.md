## 二、Sys-0 安全层测试 (小芳)

### TC-001: 急停响应
| 项 | 内容 |
|------|------|
| **输入** | 触发急停信号 |
| **预期** | Sys-0 立即返回 `SAFE=False`, 塔灯变红 |
| **验证** | `safety.ok_to_move() == False` |
| **算力** | <0.1ms (CPU) |

### TC-002: 力超阈值保护
| 项 | 内容 |
|------|------|
| **输入** | Fz=8N (超过5N阈值) |
| **预期** | Sys-0 阻止运动, 记录日志 |
| **验证** | `_check_force_limits() == False` |

### TC-003: 关节限位
| 项 | 内容 |
|------|------|
| **输入** | J1=5.0 rad (超过±3.0) |
| **预期** | 运动被阻止 |
| **验证** | `_check_joint_limits() == False` |

### TC-004: 自动重试
| 项 | 内容 |
|------|------|
| **输入** | 力异常→3次重试 |
| **预期** | 3次后转入FAULT状态 |
| **验证** | `safe_retry() == False, level==FAULT` |

### TC-005: 仿真桥精度
| 项 | 内容 |
|------|------|
| **输入** | J6逆时针10° (-0.1745rad) |
| **预期** | 运动完成后J6≈-0.8725 rad |
| **验证** | 位置偏差 <0.001 rad |

### TC-006: 传感器数据完整性
| 项 | 内容 |
|------|------|
| **输入** | 仿真桥100次更新 |
| **预期** | 5类传感器全部有数据输出 |
| **验证** | joint/force/camera/gripper/tactile 非空 |
