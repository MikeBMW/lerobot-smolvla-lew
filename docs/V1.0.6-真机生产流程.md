# Z-MAX 真机生产流程 · V1.0.6

> 基于Orin真机状态机 | 更新日期: 2026-07-12

## 一、实际生产流程

```
初始化循环 → 取料 → 扫码 → 尝试插入×2 → 等待测试 → 拔出 → AOI×6 → OK/NG → 循环
```

## 二、关键参数

| 参数 | 值 | 说明 |
|:--|:--|:--|
| 抓取力阈值 | 5N | 低于5N=抓取失败 |
| 连续抓取失败 | ≥2次 | 触发报警 |
| 插入深度 | >0.75 | end_pose[0]判断完成 |
| AOI检测面 | 6面 | 任意面fail→NG |
| NG分拣位 | 2个 | manual_pose_25/26交替 |
| 视觉服务器 | 192.168.23.26 | Windows PC |
