# Orin 真机 · 抓取放置工序 (100G AOI)

> 2026-07-12 · 小芳现场采集

## 完整流程

```
初始化循环
  ↓
取料 (触觉力检测抓取力)
  ↓
扫码
  ↓  [grasp_normal_force < 5N?]
  ├─ 是 → 抓取失败开爪 → 循环守卫
  │        [连续2次失败?]
  │         └─ 是 → 连续抓取失败报错 (STOP)
  └─ 否 ↓
尝试插入第一次 (力控 >1kHz)
  ↓  [end_pose.x > 0.750?]
  ├─ 是 → 插入完成
  └─ 否 → 尝试插入第二次 → 插入完成/失败
  ↓
等待测试结果 (HMI触发)
  ↓
拔出
  ↓  [test_result?]
  ├─ PASS → AOI_1→2→3→4→5→6 → OK/NG 分拣
  └─ FAIL → NG 分拣 → 循环守卫
  ↓
循环守卫 → 取料 (下一轮)
```

## 关键数据

| 参数 | 含义 |
|------|------|
| grasp_normal_force | 触觉传感器抓取力 |
| grasp_failed | 力<5N判定抓取失败 |
| consecutive_grasp_fail_count | 连续失败计数(>2报错) |
| end_pose.x > 0.750 | 插入到位深度判定 |
| aoi_fail_flag | AOI各工位失败累计 |
| ng_place_count | NG料盘位置轮换 |

## 容错设计

| 场景 | 策略 |
|------|------|
| 抓取力不足 | 开爪重试, 3次报错 |
| 插入不到位 | 二次尝试 |
| AOI单工位失败 | 记录标志位, 继续后续工位 |
| 最终有NG标志 | 分拣到NG料盘 |
| 循环控制 | pick_place_loop_done 信号 |

## Sys-0 安全对接点

每个运动节点都需经过:
1. 急停检查
2. 力传感器阈值 (Fz>5N)
3. 关节限位 (±3rad)
4. 光幕检测
