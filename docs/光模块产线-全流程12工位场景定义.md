## 第一阶段：光器件封装 (光学前端)

### 工位1: 晶圆上料 (Wafer Loading)
- **输入:** 4英寸 InP/GaAs 晶圆
- **操作:** 机器人从晶圆盒取晶圆 → 放置到划片机 → 视觉对位 → 真空吸附固定
- **精度:** ±5μm
- **节拍:** 30秒/片
- **3D动画:** 机器人双臂从左侧料架取圆形晶圆,旋转90°放置到设备平台

### 工位2: 共晶焊接 (Eutectic Bonding)
- **输入:** 切割后的光芯片 (LD激光器/PD探测器/EML电吸收调制激光器)
- **操作:** 取芯片 → 放置到共晶台 → 视觉粗对准(±10μm) → 金锡焊片放置 → 加热至320°C → 力控精压(±1μm, 0.5N) → 冷却固化
- **检测:** 热电偶测温 ±2°C, 视觉检查焊料溢出
- **节拍:** 15秒/颗
- **3D动画:** 机器人右臂从芯片盘取微小芯片(红光标记),放置到加热平台,热气升腾效果

### 工位3: COC贴装 (Chip on Carrier)
- **输入:** 共晶完成的芯片+陶瓷基板
- **操作:** 点银浆(视觉定位焊盘±5μm) → 贴装芯片(力控0.3N) → UV固化 → 目检银浆溢出
- **输出:** COC组件 (芯片已固定在基板上)
- **节拍:** 20秒/颗
- **3D动画:** 点胶针头挤出银色胶点,芯片从上方精确落下贴合

### 工位4: 引线键合 (Wire Bonding)
- **输入:** COC组件
- **操作:** 放置到打线机 → 视觉定位焊盘 → 超声波金丝键合(25μm金丝, 100kHz超声) → 线弧高度检测 → Next焊盘
- **检测:** 拉力测试(>5g), 线弧高度<150μm, 焊点形貌检查
- **节拍:** 30秒/颗 (6-8根金丝)
- **3D动画:** 劈刀快速下压,金色丝线从芯片电极连接到基板焊盘,形成弧形

## 第二阶段：光引擎集成 (OE Assembly) ✨核心

### 工位5: OE光引擎组装 (Optical Engine Assembly)
- **输入:** 打线完成的COC + 透镜阵列 + 隔离器 + 分光棱镜
- **流程:**
  1. 放置COC到OE基座 → 螺丝固定(0.3Nm力矩)
  2. 透镜阵列安装 → 视觉主动对准(±0.5μm) → UV固化
  3. 光隔离器插入光路 → 角度调节(<0.1°)
  4. 分光棱镜/波分复用器安装 → 5轴微调平台
  5. 光纤阵列(FA)对接 → 六维力控插入(0.05N)
- **检测:** 光功率计实时读数, 插入损耗<2dB, 回波损耗>50dB
- **输出:** 完整OE光引擎 (含光芯片+透镜+隔离器+FA光纤)
- **节拍:** 120秒/个 (最复杂的工序)
- **3D动画:** 
  - COC从左侧滑入OE基座
  - 多层透镜依次叠加(半透明效果)
  - 蓝色激光光束从芯片射出穿过透镜→隔离器→棱镜→光纤
  - 光功率计数字跳动(实时dBm显示)

## 第三阶段：电学集成

### 工位6: PCBA焊接 (Printed Circuit Board Assembly)
- **输入:** OE光引擎 + 驱动PCB板
- **操作:** OE引脚与PCB焊盘对齐 → 回流焊(260°C峰值) → 或者FPC柔性连接 → 金手指插接
- **检测:** 电气导通测试, 阻抗匹配(50Ω±5%)
- **节拍:** 45秒/块
- **3D动画:** PCB板从上方降下,焊点逐个亮起(红色→冷却变银)

### 工位7: 模块壳体装配 (Module Housing)
- **输入:** PCBA组件 + 金属壳体(CFP2/QSFP-DD/OSFP)
- **操作:** 放置导热垫 → 插入PCBA → 合盖 → 螺丝锁紧(4颗, 0.5Nm) → 安装散热片
- **拉环:** 安装解锁拉环 → 弹簧测试(10N拉力)
- **节拍:** 30秒/个
- **3D动画:** 壳体从两侧合拢,螺丝刀旋转拧紧,金色散热片装上

### 工位8: 光纤接头安装 (Optical Connector)
- **输入:** 组装完成的模块 + LC/MPO光纤跳线
- **操作:** 清洁光纤端面(酒精擦拭) → 端面检测(200×放大) → 插入LC连接器 → 锁紧
- **检测:** 端面洁净度(IEC 61300-3-35标准), 无划痕/污染
- **节拍:** 20秒/个
- **3D动画:** 蓝色光纤从右侧插入,锁扣咔嗒闭合

## 第四阶段：测试与老化

### 工位9: 模块老化 (Burn-in)
- **输入:** 组装完成的光模块
- **操作:** 插入老化板(16通道/板) → 放入老化箱 → 设定85°C/85%RH → 上电(3.3V/5V) → 72小时监控
- **监控:** 每15分钟记录光功率/温度/偏置电流
- **节拍:** 5秒/模块 (操作), 72小时 (老化)
- **3D动画:** 模块一个个插入老化板绿色插槽,箱门关闭,温度显示器上升到85°C,内部红色光点闪烁

### 工位10: 终检测试 (Final Test)
- **输入:** 老化完成的光模块
- **测试项目:**
  1. 光眼图测试 (Eye Diagram) → 消光比>4dB, 抖动<0.3UI
  2. 误码率测试 (BERT) → BER<1E-12 @ 53.125Gbaud
  3. 光谱测试 → 中心波长±1nm, 边模抑制比>30dB
  4. 接收灵敏度 → <-10dBm @ 100G per lane
  5. 功耗测试 → <12W (QSFP-DD 400G)
- **设备:** 采样示波器+光功率计+光谱仪+BERT
- **节拍:** 60秒/模块
- **3D动画:** 模块插入测试板,示波器显示眼图张开,绿色PASS灯亮起

### 工位11: 打标分选 (Marking & Sorting)
- **输入:** 测试完成的模块
- **操作:** 激光打标(二维码+序列号) → 良品/不良品分选 → 静态袋真空包装 → 贴标签
- **不良品:** 送返修站(拆解→重焊→复测)
- **节拍:** 15秒/个
- **3D动画:** 激光在模块表面刻出二维码,机械臂将良品放入绿色框,不良品放入红色框

### 工位12: 包装出货 (Packaging)
- **输入:** 已打标的良品模块
- **操作:** 放入防静电托盘 → 干燥剂+湿度卡 → 真空包装 → 装箱 → 贴外箱标签
- **输出:** 成品光模块, 可出货
- **节拍:** 20秒/个
- **3D动画:** 托盘从上方堆叠,真空机吸气封口,箱子自动封箱
