## 一、行业标准对照 · L1~L5

| 等级 | 名称 | 定义 | 决策主体 | 人工角色 | Z-MAX 产品对应 |
|------|------|------|----------|----------|---------------|
| **L1** | 辅助操作 | 单一功能辅助，人做主要动作 | 人 | 全程参与操作 | — |
| **L2** | 分段式自动化 | 预设工序自动执行，人监控和切换 | 人编排规则 | 编排+监控+异常处理 | **Z700 F 基线版** |
| **L3** | 条件自动化 | 传感器+视觉自动适应，人处理少数异常 | 机器+人协同 | 异常处理 | Z700 增强版 |
| **L4** | 高度自动化 | 全自主执行+自适应+自恢复 | 机器 | 仅极限情况介入 | Z700 旗舰版 |
| **L5** | 完全自主 | 任何场景无人工干预，自我学习进化 | 机器 | 零参与 | 未来目标 |

## 二、Z-MAX 产品等级详细定义

### L2 基线版 — Z700 F · Sys-0 分段式

| 维度 | 说明 |
|------|------|
| **核心能力** | 人工流程编排 + 标准原子功能库 + 分段式验证 |

| **软件引擎** | Sys-0 人工编排 · ROS2 Service 接口库 |
| **算力平台** | Jetson Orin Nano 8GB (40 TOPS) |
| **操作流程** | 6步: 人工编排→原子功能→动作执行→力控反馈→AOI验证→下料 |
| **视觉** | 人工扫码 · 预设位姿 |

| **硬件** | SR5-C机械臂 + Orin Nano + 双3D + DH夹爪 + TS-T-15 |

| **ROS2** | 21节点 · 51Topic · Domain 23 |
| **状态** | ✅ 已交付 · 苏州实验室 Phase 0 验收通过 |
