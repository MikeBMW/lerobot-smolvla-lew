## 一、产品定位

**Z700 F 是一台固定式精密操作机器人，专为光模块产线精细插拔工序设计。**

它搭载 Z-MAX 智能化模型引擎，在 L2 基线版采用人工编制流程 + 标准原子功能库的方式，实现±0.05mm 精度的力控插拔操作。操作员通过 XSpace Studio 界面编排工序流程，机器人自动执行每个原子功能。

Z700 F 是 Z700 系列的首个量产版本。未来通过 OTA 软件升级可演进至 L3 增强版和 L4 旗舰版，硬件无需更换。

## 三、L2 基线版 · 操作流程

L2 采用 **人工编制流程 + 标准原子功能库** 的工作方式：

```
操作员编排 ──→ 标准原子功能 ──→ 动作执行 ──→ 力控反馈 ──→ 分段验证 ──→ 成品下料
```

| 步骤 | 名称 | 说明 |
|------|------|------|
| 1 | 人工流程编排 | 操作员在 XSpace Studio 中设定工序参数，选择标准原子功能 |
| 2 | 标准原子功能 | 取料、扫码、定位、对准、插入、拔出、检测、分类 |
| 3 | 动作执行 | 基于 ROS2 标准接口，点到点精确运动 ±0.05mm |
| 4 | 力控反馈 | 六维力传感器 >10kHz 采样，夹持力自适应闭环 |
| 5 | 分段式验证 | 每步完成确认后才进入下一步 · 异常自动停机 |
| 6 | 成品下料 | 取出完成品并记录检测数据 |

## 九、日常操作

```
开机: 按电源 → 等绿灯亮 → 打开 XSpace Studio
选工序: 固件加载 / 老化上下料 / 模块测试
点"开始" → Z700 F 自动执行
```

**塔灯颜色说明:**
- 🟢 绿灯 = 正常运行
- 🟡 黄灯 = 需要人工处理（缺料/模块异常）
- 🔴 红灯 = 紧急停止（排除障碍后复位）

**常见问题处理:**

| 现象 | 操作 |
|------|------|
| 黄灯亮 | 检查缺料 → 补充模块 → 点"继续" |
| 夹爪抓不住 | 点"重新定位" |
| 红灯亮 | 排除障碍 → 拧开急停 → 点"复位" |
| 测试不通过 | 模块放入不良品盒 → 继续下一个 |
