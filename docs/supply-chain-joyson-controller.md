## 二、控制器硬件方案

### 2.1 域控制器 (Domain Controller)

均胜的域控制器基于异构SoC架构，适用于Z-MAX机器人中央计算平台：

| 参数 | 规格 | 说明 |
|------|------|------|
| **主SoC** | 高通SA8295/Orin系列 | 算力30-200+ TOPS |
| **MCU** | Infineon TC3xx/TC4xx | 实时控制 (ASIL-D) |
| **内存** | LPDDR5 8-32GB | 大模型推理 |
| **存储** | UFS 3.1 128-512GB | 数据集+模型 |
| **接口** | 多路GMSL/Ethernet/CAN-FD | 传感器+执行器 |
| **系统** | QNX + Linux (Hypervisor) | 功能安全+AI |

### 2.2 电源管理系统 (BMS/PDU)

| 参数 | 规格 | Z-MAX适配 |
|------|------|----------|
| 输入电压 | 48V/400V/800V | Z700电池组 |
| 功率输出 | 500W-5kW | 双臂+底盘 |
| 效率 | >95% | 延长续航 |
| 保护 | 过压/过流/过温/短路 | 工业安全 |

### 2.3 传感器融合模块

| 传感器 | 接口 | 数据速率 |
|--------|------|---------|
| 摄像头 | GMSL2 (6Gbps) | 8路×30fps |
| 激光雷达 | 100Base-T1 | 1路 |
| 毫米波雷达 | CAN-FD | 4-6路 |
| IMU | SPI | 1路 200Hz |
| 力传感器 | EtherCAT | 6路 1kHz |

## 三、Z-MAX控制器硬件选型建议

### 方案A: 基于均胜域控制器 (推荐)

```
硬件平台: Joyson DCU Gen3
  ├── 主SoC: NVIDIA AGX Orin (275 TOPS)
  ├── 安全MCU: Infineon TC397 (ASIL-D)
  ├── 内存: 32GB LPDDR5
  ├── 接口:
  │   ├── GMSL2 ×8 → 相机
  │   ├── EtherCAT ×2 → 力传感器/关节编码器
  │   ├── CAN-FD ×4 → 夹爪/底盘
  │   ├── 10GbE ×2 → 网络
  │   └── GPIO ×16 → IO控制
  └── 供电: 48V输入 → 12V/5V/3.3V/1.8V多路输出
```

### 方案B: 自研控制器 + 均胜模块

```
主控: Z-MAX定制主板
  ├── 算力: 均胜DCU模块 (Orin NX 16GB)
  ├── 电源: 均胜BMS模块 (48V→多路)
  ├── 安全: 均胜Safety MCU (TC397)
  └── 接口: 自研IO板 (EtherCAT/CAN/RS485)
```
