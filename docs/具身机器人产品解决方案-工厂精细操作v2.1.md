## 三、插拔工作流程（双臂协同）

### 3.1 左臂动作（取料与扫码）

1. 左臂腕部视觉识别散堆来料，引导执行机构吸取
2. 调整腕部姿态，移至扫码器位置扫码
3. 将物料按标准姿态放入中转区
4. 循环取料，保持物料中转区供给

### 3.2 右臂动作（插拔与分类）

1. 右臂视觉识别中转区物料，夹爪抓取
2. 运动至EVB测试工位，视觉对准 + 力控插入
3. 再次从中转区抓取（双工位并行）
4. 运动至第二工位插入
5. 左工位测试完成 → 拔出 → 送入AOI检测
6. AOI完成 → 抓取 → 补料左工位
7. 依据检测结果分类放置至合格/NG物料盒
