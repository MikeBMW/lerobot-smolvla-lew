## 二、六大工位详细设计

### 工位1: 共晶 (Eutectic Bonding)
- **任务:** 将光芯片 (LD/PD) 与基板通过金锡焊料精确焊接
- **精度要求:** ±1μm 贴装, ±5°C 温控
- **机器人操作:**
  - 从料盘取光芯片 (真空吸嘴)
  - 放置到共晶台上预对准
  - 视觉引导精对准 (双目微距相机, 0.5μm 分辨率)
  - 力控压合 (0.1N 精度, 避免芯片碎裂)
  - 完成信号反馈给MES系统
- **节拍:** 15秒/颗

### 工位2: COC贴装 (Chip on Carrier)
- **任务:** 将共晶完成的芯片贴装到陶瓷基板
- **机器人操作:**
  - 从共晶台取已完成芯片
  - 点胶 (视觉引导定位, ±5μm)
  - 贴装到COC基板指定位置
  - UV固化 + 力控保压 (0.5N, 3秒)
- **节拍:** 20秒/颗

### 工位3: 打线 (Wire Bonding)
- **任务:** 金丝键合, 连接芯片电极到基板焊盘
- **机器人操作:**
  - 将COC组件放置到打线机夹具
  - 视觉检查焊盘位置 (±2μm)
  - 启动打线程序 (超声+热压)
  - 打完线后取件 → 视觉质检 (检查线弧高度、焊点形貌)
- **节拍:** 30秒/颗 (含质检)

### 工位4: 上下料 (Loading/Unloading)
- **任务:** 在共晶机/打线机/老化箱之间转运物料
- **机器人操作:**
  - 从上游工位取Tray盘 (双臂协同, 防倾斜)
  - 沿轨道移动至下游工位 (移动速度 0.5m/s)
  - 将Tray盘放置到设备进料口
  - 扫码确认批次信息 (手腕集成条码枪)
- **节拍:** 8秒/次搬运

### 工位5: 老化箱 (Burn-in Oven)
- **任务:** 85°C/85%RH 环境下72小时老化测试
- **机器人操作:**
  - 从打线工位取待老化模块
  - 打开老化箱门 (力控开门, 门重约5kg)
  - 将模块插入老化板 (视觉对准插座, ±0.2mm)
  - 关闭箱门 → 记录开始时间
  - 72小时后取出 → 送测试工位
- **节拍:** 5秒/模块 (插拔)

### 工位6: 终检测试
- **任务:** 光功率/眼图/误码率测试
- **机器人操作:**
  - 从老化箱取模块 → 插入测试板
  - 光纤自动对准 (视觉+微动平台, ±0.5μm)
  - 启动测试程序 → 等待结果
  - 良品/不良品分拣至不同Tray盘
- **节拍:** 45秒/模块 (含测试时间)

## 三、关键参数

| 参数 | 数值 | 对标设备 |
|:--|:--|:--|
| 移动速度 | 0.5m/s | 轨道驱动 |
| 定位精度 | ±0.01m | 全向底盘 |
| 重复精度 | ±0.1mm | 7-DOF机械臂 |
| 力控精度 | 0.1N | 六维力传感器 |
| 视觉精度 | 0.5μm | 双目微距 |
| 单臂负载 | 5kg | 双臂协同 |
| 循环时间 | <5分钟/模块 | 全流程 |

## 五、成本优势

- 1台人形机器人替代3-4个操作员 (24h连续作业)
- 轨道方案节省AGV成本 (无电池/充电需求)
- 视觉质检替代人工目检, 良率从97% → 99.5%
- 年产能: ~200万颗/年 (单线, 按5分钟/颗计)
