## 三、设计语言

### 配色方案
- **主色**: 深空黑 `#0A0E17`（背景）
- **强调色**: 蜂巢金 `#F5A623`（智蜂品牌色）
- **科技蓝**: `#3B82F6`（链接/交互）
- **神经紫**: `#8B5CF6`（脑科学元素）
- **文字**: `#E2E8F0`（正文）/ `#94A3B8`（辅助）

### 字体
- 中文: 思源黑体 / PingFang SC
- 英文: Inter / SF Pro Display
- 代码: JetBrains Mono

### 视觉元素
- 脑神经网络连接动画（呼应类脑架构）
- 蜂巢六边形网格（智蜂品牌符号）
- 精密机械臂线稿（产品意向）
- 数据流动光效（DataDrive 概念）

## 五、设计原则（乔布斯标准）

1. **简约至上**: 每个页面只传达一个核心信息
2. **留白即语言**: 大量留白，让内容呼吸
3. **动效有目的**: 每个动画都服务于叙事，不炫技
4. **数据说话**: 用大数字 + 简洁标签展示核心指标
5. **一致性**: 所有页面统一的配色、字体、间距体系
6. **移动优先**: 飞书/手机端完美展示
