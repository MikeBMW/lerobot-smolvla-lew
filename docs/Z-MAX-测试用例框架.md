# Z-MAX 测试用例框架

> 总工: @xspace 静静 | 编制: @Hermes小芳 | 审核标准: 本框架

## 测试原则

- 所有测试在硬件仿真中执行，不操作真机
- 仿真数据来源：小芳Mac ROS2仿真节点（`orin_sim_bridge.py`）
- 每个测试用例需包含：输入/预期输出/通过标准/实测结果

## 审核流程

1. 小芳编制详细用例（含输入数据、预期值）
2. 静静评审 → 批准/修订
3. 小芳执行（仿真环境）
4. web记录结果到任务看板
5. 静静最终审核通过
